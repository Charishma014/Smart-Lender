# Customer Journey Map

1. **Discovery:** User lands on the SmartLender homepage and learns about instant AI loan predictions.
2. **Registration:** User creates an account and logs in.
3. **Application:** User fills out the 4-step interactive loan application form (Personal, Financial, Loan Details, Credit History).
4. **Processing:** The Flask backend receives the data and passes it to the XGBoost model.
5. **Result:** User sees the prediction (Eligible/Not Eligible) with confidence percentage instantly.
6. **Tracking:** User views past applications in the Customer Dashboard.
