# Proposed Solution

We propose **SmartLender**, an integrated web application that serves as a bridge between machine learning and end-users. 

The system utilizes a pre-trained XGBoost classifier wrapped in a Flask web application. When a user submits their financial and personal details, the application instantly evaluates the risk and eligibility. Furthermore, the platform acts as an educational hub, providing users with information on loan types, interest rates, and ways to improve their credit scores.
