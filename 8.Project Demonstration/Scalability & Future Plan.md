# Scalability & Future Plan

## Scalability
- **Cloud Deployment:** Migrate from local Flask server to AWS/Heroku/GCP.
- **Database:** Move to MongoDB Atlas for a managed, scalable database cluster.
- **Containerization:** Dockerize the application for consistent deployment environments.

## Future Enhancements
- **Advanced AI Models:** Integrate Deep Learning models if the dataset grows significantly.
- **Real-time Credit Check:** Integrate with third-party credit bureau APIs (e.g., Experian, Equifax).
- **Gen-AI Chatbot:** Add an AI assistant to answer user queries about their loan application.
