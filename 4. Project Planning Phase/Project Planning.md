# Project Planning

## Phases and Milestones

- **Phase 1: Research & Setup (Days 1-2)**
  - Define requirements, select tech stack, setup repository and virtual environment.

- **Phase 2: Data & ML Model (Days 3-5)**
  - Generate/acquire dataset, perform EDA (`eda.py`), train XGBoost model (`train_model.py`), and export pipeline.

- **Phase 3: Backend Development (Days 6-8)**
  - Setup Flask app (`app.py`), connect to MongoDB (`database.py`), implement authentication and `/api/predict` endpoint.

- **Phase 4: Frontend Development (Days 9-12)**
  - Design UI/UX, build HTML templates, style with CSS, implement 4-step wizard.

- **Phase 5: Integration & Testing (Days 13-14)**
  - Connect frontend to backend, test ML predictions, fix bugs, finalize documentation.
