# Brainstorming & Idea Prioritization

## Brainstorming Ideas
1. **Manual Loan Processing:** Traditional banks take weeks to process loan applications. We need a faster solution.
2. **AI-Powered Decision Making:** Use Machine Learning to predict loan eligibility instantly based on user data.
3. **Educational Platform:** Not just predict, but also educate users about loan types, interest rates, and CIBIL scores.

## Idea Prioritization
- **High Priority:** Build the core ML model (XGBoost) and integrate it with a web interface (Flask) for instant predictions.
- **Medium Priority:** User authentication, dashboards for tracking history, and MongoDB integration.
- **Low Priority:** Admin dashboard for analytics (to be added once core is stable).
