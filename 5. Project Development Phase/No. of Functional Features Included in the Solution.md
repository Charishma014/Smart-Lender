# Functional Features Included

1. **User Authentication:** Secure Sign Up and Login.
2. **Multi-step Loan Application Form:** Interactive UI for data entry.
3. **Instant ML Prediction:** Real-time eligibility checking.
4. **Confidence Scoring:** Displays the model's confidence in its decision.
5. **Customer Dashboard:** View history of all submitted applications.
6. **Admin Dashboard:** Overview of total applications, approval rates, and analytics.
7. **Loan Information Hub:** Educational resources for users.
8. **REST API Endpoint:** `/api/predict` for programmatic access.
